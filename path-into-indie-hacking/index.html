In autumn 2019 I started thinking about different product ideas and noticed all talk and no walk will not get me anywhere.

That is when I started learning about building products and how to actually create things for the web. I started out learning as much as I can about all kinds of things like NoCode Tools, UX, UI, read books and enrolled in all kinds of online classes, communites, forums, subreddits and, of course, Youtube rabbit holes.

It was during one of my Youtube Coding / Lifesytle Blogger rabbit hole loops where the Youtuber and programmer with the name of Aaron Jack mentioned Pieter Levels.

After I watched his video on how to start a startup and hearing his bold approach towards creating things I was really motivated.

" The biggest thing in coding and in business is to learn how to lean and to learn how to figure things out for yourself “ - Pieter Levels

It was then in the Summer of 2020 where I discovered the online community indie hackers and I was blown away.

A whole community of people trying to achieve exactly what I was trying to do.

So many groups and people discussing topics I was loosely trying to put together in my asana to do list.

Especially the No Code Movement grabbed my attention as I know basic web developmemnt concepts, but can not code whole products of my own.

I loved the way in which information got shared and everyone supported each other.

After working with all kind of No Code tools I came to one realization:

I want to have a better understanding of whats happening under the hood.

No Code is great and todays platforms are always getting even better, but at some point I saw features where I knew “this is one line of html” and I had to pay 5$ extra just to activate the feature.

This is when I made the decision I do not want to be dependent on tools, but rather have the freedom of building on my own terms.

And this is where I am right now.

I am still in the process of learning coding concepts but also using No Code tools to be fast with certain things.

My goal is to create and learn as much as possible in 2021.
